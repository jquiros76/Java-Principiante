Variables & Tipos de Datos

¿Qué es una variable?

Una variable es un espacio en memoria donde guardas un valor. En Java debes declarar el tipo antes del nombre. Java es un lenguaje fuertemente tipado: no puedes mezclar tipos sin conversión.

📌 Sintaxis
tipo nombreVariable = valor;

Tipos Primitivos

TipoTamañoEjemploUso
int32 bitsint edad = 25;Números enteros
double64 bitsdouble precio = 9.99;Decimales
boolean1 bitboolean activo = true;true / false
char16 bitschar letra = 'A';Un carácter
long64 bitslong big = 9999999L;Enteros grandes
float32 bitsfloat n = 3.14f;Decimal menos preciso

String — Tipo Referencia

String no es un tipo primitivo, es una clase. Se escribe con mayúscula y se usa para texto.

Main.java
public class Main {
    public static void main(String[] args) {

        // Variables primitivas
        int edad = 22;
        double salario = 1500.50;
        boolean activo = true;
        char inicial = 'J';

        // Variable de tipo referencia
        String nombre = "Carlos";

        // Imprimir en consola
        System.out.println("Nombre: " + nombre);
        System.out.println("Edad: " + edad);
        System.out.println("Activo: " + activo);
    }
}

Constantes con final

Usa final para declarar un valor que no puede cambiar. Por convención se escribe en MAYÚSCULAS.

Constantes.java
final double PI = 3.14159;
final int MAX_USUARIOS = 100;
// PI = 3.0; ← ERROR: no se puede reasignar
💡 TIP
Usa nombres descriptivos en camelCase: nombreUsuario, totalVentas. Evita: x, a1, temp.

Ejemplo completo

Operadores.java
int a = 10, b = 3;

System.out.println(a + b);   // 13
System.out.println(a - b);   // 7
System.out.println(a * b);   // 30
System.out.println(a / b);   // 3  ← división entera
System.out.println(a % b);   // 1  ← resto

boolean mayor = (a > b);      // true
boolean ambosPos = (a > 0 && b > 0); // true

// Incremento
a++;  // a = 11
b--;  // b = 2

// Asignación compuesta
a += 5;  // a = 16
b *= 3;  // b = 6
⚠️ CUIDADO
10 / 3 = 3 en Java (no 3.33). Para decimales usa: (double) 10 / 3 → 3.333...

TEMA 03 / 06
Condicionales

if / else if / else

Permiten ejecutar bloques de código según una condición booleana.

Condicional.java
int nota = 75;

if (nota >= 90) {
    System.out.println("Excelente");
} else if (nota >= 70) {
    System.out.println("Aprobado");
} else if (nota >= 60) {
    System.out.println("Regular");
} else {
    System.out.println("Reprobado");
}

switch

Ideal cuando comparas una variable contra valores exactos.

Switch.java
String dia = "LUNES";

switch (dia) {
    case "LUNES":
        System.out.println("Inicio de semana");
        break;
    case "VIERNES":
        System.out.println("¡Fin de semana!");
        break;
    default:
        System.out.println("Día normal");
}

Operador Ternario

Forma compacta de if-else en una línea.

Ternario.java
int edad = 20;
String estado = (edad >= 18) ? "Mayor de edad" : "Menor de edad";
System.out.println(estado); // Mayor de edad
💡 Sintaxis
condición ? valorSiTrue : valorSiFalse

Ciclos (Loops)

for — cuando sabes cuántas veces

Ciclos.java
// Imprime del 1 al 5
for (int i = 1; i <= 5; i++) {
    System.out.println("Iteración: " + i);
}

// Tabla de multiplicar del 3
for (int n = 1; n <= 10; n++) {
    System.out.println("3 x " + n + " = " + (3 * n));
}

while — mientras se cumpla una condición

While.java
int contador = 0;
while (contador < 5) {
    System.out.println("Contador: " + contador);
    contador++;  // ¡No olvides incrementar!
}

do-while — ejecuta al menos una vez

DoWhile.java
int x = 10;
do {
    System.out.println("x = " + x);
    x++;
} while (x < 5); // Se ejecuta 1 vez aunque x >= 5

break y continue

BreakContinue.java
for (int i = 0; i < 10; i++) {
    if (i == 3) continue; // salta el 3
    if (i == 7) break;    // detiene el loop en 7
    System.out.println(i); // imprime 0,1,2,4,5,6
}

TEMA 05 / 06
Arrays (Arreglos)

¿Qué es un array?

Un array es una colección de elementos del mismo tipo, con tamaño fijo. El índice empieza en 0.

Arrays.java
// Declarar e inicializar
int[] numeros = {10, 20, 30, 40, 50};

// Acceder por índice
System.out.println(numeros[0]); // 10
System.out.println(numeros[4]); // 50

// Longitud
System.out.println(numeros.length); // 5

// Modificar elemento
numeros[2] = 99; // {10, 20, 99, 40, 50}

// Recorrer con for-each
for (int n : numeros) {
    System.out.println(n);
}

Array de Strings

ArrayStrings.java
String[] nombres = {"Ana", "Luis", "María"};

for (int i = 0; i < nombres.length; i++) {
    System.out.println("Hola, " + nombres[i]);
}
⚠️ ERROR COMÚN
Acceder a un índice fuera del rango genera ArrayIndexOutOfBoundsException. Si el array tiene 5 elementos, los índices válidos son 0–4.

TEMA 06 / 06
Métodos

¿Qué es un método?

Un método es un bloque de código reutilizable que realiza una tarea. En Java todo método pertenece a una clase.

📌 Anatomía
modificador tipoRetorno nombreMetodo(parámetros) { ... }

Método sin retorno (void)

Metodos.java
public class Metodos {

    // Método void — no retorna nada
    public static void saludar(String nombre) {
        System.out.println("Hola, " + nombre + "!");
    }

    // Método con retorno int
    public static int sumar(int a, int b) {
        return a + b;
    }

    // Método con retorno String
    public static String clasificar(int nota) {
        return (nota >= 70) ? "Aprobado" : "Reprobado";
    }

    public static void main(String[] args) {
        saludar("Carlos");          // Hola, Carlos!
        int res = sumar(5, 8);      // 13
        System.out.println(res);
        System.out.println(clasificar(85)); // Aprobado
    }
}

Sobrecarga de métodos (Overloading)

Mismo nombre, diferentes parámetros. Java elige cuál usar según los argumentos.

Overload.java
public static int    area(int lado)            { return lado * lado; }
public static int    area(int ancho, int alto) { return ancho * alto; }
public static double area(double radio)         { return 3.14159 * radio * radio; }

// Java elige automáticamente
area(5);           // → 25
area(4, 6);        // → 24
area(3.0);         // → 28.27
